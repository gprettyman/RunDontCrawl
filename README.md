# RunDontCrawl
A "dungeon crawler" style game that I hope will be fun.

Core Objectives of the game:
  1) Fun
  2) Simple

Core Components:
  1) Fighting System
  2) Dungeon Exploration
  3) Level Up System
  4) Shop System

Features to be added:
  1) Procedurally generated
  2) Class system or Skill Choices
